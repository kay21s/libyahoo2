CVS Information

You can get the latest source code by anonymous cvs:

cvs -z3 -d:pserver:anonymous@cvs.libyahoo2.sourceforge.net:/cvsroot/libyahoo2 co libyahoo2

This will create a directory called libyahoo2 in your current working directory.

To update your cvs from within the libyahoo2 directory:

cvs update

If you find bugs in the releases, try the CVS, some bugs are fixed in here.

To build the code from CVS, first run the gen script. Then proceed with the normal configure, make, make install process.
